# AI API Projects
Learning the Anthropic/Claude API through hands-on projects.

## Topics Covered

- ✅ Basic Chat & Multi-turn conversations
- ✅ System prompts
- ✅ Response streaming
- ✅ Structured data
- ✅ Prompt evaluation
- ✅ Prompt engineering
- ✅ Tool use with Claude (implemented with OpenRouter/Qwen)
- ✅ RAG and Agentic Search
- ✅ Features of Claude
- ✅ Model Context Protocol (MCP)
- ⬜ Agents & workflows


## Tech Stack
- Python, Jupyter Notebooks
- Groq API (LLaMA model)
- OpenRouter API (Qwen 2.5 72B)
- Anthropic Claude API
  
## Setup
1. Clone the repo
2. `pip install groq anthropic openai python-dotenv`
3. Create `.env` file with your API keys:
   - `GROQ_API_KEY=your_key`
   - `OPENROUTER_API_KEY=your_key`
   - `VOYAGE_API_KEY`
4. Run any notebook
